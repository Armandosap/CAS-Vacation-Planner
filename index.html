Connect to JSONbin.io (free) so every team member who opens this file reads and writes to the same database. All changes — names, vacations, status — are saved automatically.

1
Go to jsonbin.io, create a free account and sign in.
2
Click + Create Bin. Paste as initial content and save:
{"members":[],"vacations":[]}
The Bin ID appears in the URL after /b/ — copy it.
3
In the left menu click API Keys → + CREATE KEY → Master Key. Copy the key (starts with $2a$10$).
4
Paste both values below, click Save & Connect. Share this HTML file with the team.
